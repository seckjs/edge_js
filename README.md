# Chaoxing Edge Assistant Extension

## 项目说明

本项目主要面向超星学习通教师端，提供课程识别、作业提取、提交记录整理和主观题辅助批改能力，帮助教师减少重复操作，提高批阅效率。

## 版本信息

- 当前版本：`1.1.0`
- 版本定位：基于 `1.0.0` 的首次功能增强版本
- 版本建议：本次更新包含新能力接入、界面调整和文案统一，适合使用语义化版本中的次版本升级

## 1.1.0 更新说明

- 新增插件网关模式，支持固定网关地址、插件授权码和短期会话校验。
- 保留自定义 API 模式，兼容原有用户自定义接口的使用方式。
- 优化弹窗和页面内批阅面板样式，整体改为更克制、更自然的黑白灰主调配色。
- 统一用户可见文案，主要界面与提示信息改为中文。
- 完善批量批阅前的配置校验，减少因授权或参数缺失导致的启动失败。

## 本地加载

1. 打开 `edge://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `edge_js/`

加载完成后，建议将扩展固定到工具栏，方便随时打开主页、日志和设置面板。

## 主要文件

- `manifest.json`：扩展入口、权限、站点匹配和资源声明
- `background.js`：后台消息分发、下载处理、权限处理和网关会话逻辑
- `content.js`：页面识别、数据抓取、题目解析和批阅面板逻辑
- `popup.html` / `popup.js` / `popup.css`：扩展弹窗界面
- `review-shared.js`：共享配置、提供方预设和通用工具
- `page-bridge.js`：页面桥接与提交监听逻辑
- `icons/`：扩展图标资源

## 说明

- 当前扩展基于 Manifest V3
- 支持“插件网关”和“自定义 API”两种连接模式
- 修改代码后，可在扩展管理页点击“重新加载”使用最新版本
